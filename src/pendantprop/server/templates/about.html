What is PendantProp?

PendantProp is an automated system for measuring surface tension using the pendant drop method. It integrates with Opentrons liquid handling robots to enable high-throughput characterization of surfactants and formulations.

Features

  • Automated pendant drop formation and imaging
  • Real-time surface tension analysis
  • Dynamic surface tension measurements
  • Integration with Opentrons OT-2 robot
  • Environmental monitoring (temperature, pressure, humidity)
  • Web-based interface for experiment control

Measurement Workflow

  1. Initialize: Upload deck layout and configure experiment
  2. Measure: Upload sample information and start measurements
  3. Analyze: View real-time results and plots
